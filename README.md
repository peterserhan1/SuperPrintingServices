# Super Printing Services

Professional website for **Super Printing Services** — a full-service printing and display production company based in Roumieh, Lebanon.

## About

Super Printing Services specializes in:

- **Production**: Graphic Design, Digital Printing, Lamination, Die-Cutting, Assembly
- **Services**: In-Store Branding, Temporary & Permanent Display Stands, Exhibitions & Events, Outdoor Posters & Vehicle Branding

## Tech Stack

- Pure HTML5, CSS3, JavaScript (no frameworks)
- Responsive design with mobile-first approach
- Intersection Observer API for scroll animations
- CSS Grid & Flexbox layout
- Image lightbox gallery with keyboard navigation

This is a static website that can be served from any web server or hosted via GitHub Pages.
